# Garagenmietvertrag - Spezifikation

## Rechtliche Grundlage

Garagenmietverträge (separat von Wohnung) unterliegen **nicht** dem Wohnraummietrecht.
Es gilt weitgehende Vertragsfreiheit wie bei Gewerbe.

### Besonderheiten:

- Keine Mietpreisbindung
- Kaution frei vereinbar
- Kürzere Kündigungsfristen möglich (oft 1-3 Monate)
- Kein Kündigungsschutz

**Achtung:** Ist die Garage **zusammen mit Wohnung** vermietet (einheitliches Mietverhältnis), gilt das Wohnraummietrecht!

## Formular-Felder (Kompakt)

### §1 Vertragsparteien
- **Vermieter:** Name, Anschrift
- **Mieter:** Name, Anschrift, ggf. Telefon/Email

### §2 Mietobjekt
- **Art:** Garage / Carport / Stellplatz (PKW/Motorrad)
- **Adresse:** Straße, Hausnummer, PLZ, Ort
- **Lage:** Garagennummer, Stellplatznummer
- **Beschreibung:** z.B. "Einzelgarage Nr. 5, mit elektrischem Torantrieb"

### §3 Mietdauer
- **Mietbeginn:** Datum
- **Laufzeit:** Unbefristet / Befristet
- **Kündigungsfrist:** z.B. "3 Monate zum Monatsende"

### §4 Miete
- **Monatliche Miete:** € (brutto, da Kleinvermietung meist ohne USt)
- **Fälligkeit:** zum 1./3. Werktag
- **Zahlungsweise:** Überweisung/Dauerauftrag

### §5 Kaution (optional)
- **Betrag:** € (oft 1-2 Monatsmieten)
- **Art:** Überweisung, Bar, Bürgschaft

### §6 Nutzung
- Nur zum Abstellen von Kraftfahrzeugen
- Keine Lagerung von brennbaren Stoffen
- Keine gewerbliche Nutzung (oder nur mit Zustimmung)

### §7 Schlüssel
- Anzahl der Schlüssel
- Kosten bei Verlust

### §8 Haftung
- Keine Haftung des Vermieters für Diebstahl/Beschädigung (außer bei Vorsatz/grober Fahrlässigkeit)

### §9 Sonstiges
- Sonstige Vereinbarungen (Freitext)

### §10 Unterschriften
- Ort, Datum
- Vermieter, Mieter

## Formular-Steps (Vereinfacht)

Da der Garagenvertrag simpel ist, reichen **4-5 Steps**:

1. **Parteien** - Vermieter + Mieter
2. **Garage** - Art, Adresse, Nummer
3. **Konditionen** - Beginn, Miete, Kaution, Kündigung
4. **Abschluss** - Ort, Datum, Unterschrift

## Unterschiede je nach Art

| Art | Besonderheiten |
|-----|----------------|
| **Garage** | Geschlossener Raum, Schlüssel, Stromversorgung möglich |
| **Carport** | Überdacht aber offen, kein Schlüssel |
| **Stellplatz** | Freifläche, Markierung, ggf. Schranke |
| **Tiefgarage** | Schlüssel/Fernbedienung, Stellplatznummer |

## Muster-Vertragstext (Kurzfassung)

```
GARAGENMIETVERTRAG

zwischen
[Vermieter Name], [Anschrift]
- nachfolgend Vermieter -

und
[Mieter Name], [Anschrift]
- nachfolgend Mieter -

§1 Mietgegenstand
Der Vermieter vermietet dem Mieter die [Garage/Stellplatz] Nr. [X]
in [Adresse].

§2 Mietzeit
Das Mietverhältnis beginnt am [Datum] auf unbestimmte Zeit.
Es kann von beiden Seiten mit einer Frist von [X] Monaten 
zum Monatsende gekündigt werden.

§3 Miete
Die monatliche Miete beträgt [X] EUR.
Sie ist bis zum [3.] Werktag eines Monats auf folgendes Konto zu überweisen:
IBAN: [IBAN]

§4 Kaution
Der Mieter zahlt eine Kaution in Höhe von [X] EUR.

§5 Nutzung
Die Garage/Der Stellplatz darf nur zum Abstellen von Kraftfahrzeugen 
genutzt werden. Die Lagerung von brennbaren oder umweltgefährdenden 
Stoffen ist untersagt.

§6 Schlussbestimmungen
Änderungen und Ergänzungen bedürfen der Schriftform.

[Ort], [Datum]

_________________          _________________
Vermieter                  Mieter
```
